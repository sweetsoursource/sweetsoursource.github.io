Chronfluence Plugin

What it does

The Chronfluence Plugin for Confluence allows you to visualize content updates, e.g. page, space and user updates, new comments, attachments in a nice timeline.

In addition it is possible to apply specific filters like space-wise or by content type and to preview the results in the macro browser.

Download Chronfluence Plugin at Atlassian Marketplace

Dashboard

After installing the plugin you will see that the Dashboard contains a new section on the left side. It will display recent updates of pages, users, comments etc. as a timeline.

Macro

You can add Chronfluence as a macro on a page by choosing the Timeline macro in the macro browser. Configuring space or content type filter you can customize the result set that Chronfluence will display in a timeline.
